# Design Taste

Use this skill when designing layout structures, spacing, color themes, or component outlines.

## Guidelines

### 1. Color and Contrast
- **Semantic Colors**: Avoid saturated primaries. Prefer rich, muted HSL/OKLCH color spectrums.
- **Glassmorphism**: Use backdrops for depth and polish: `bg-card/30 backdrop-blur-lg border border-border/35`.
- **Gradients**: Leverage soft gradient backgrounds for headers or highlight cards: `bg-gradient-to-tr from-primary/10 via-primary/5 to-transparent`.

### 2. Spacing and Geometry
- **Consistent Gaps**: Restrict element separation to strict grid gaps: `gap-4` or `gap-6`.
- **Card Padding**: Give cards room to breathe: use `p-6` or `p-8` for padding.
- **Rounded Edges**: Enforce soft corner radii: `rounded-2xl` (16px) or `rounded-3xl` (24px) for cards/dialogs.
