# UI/UX Design

Use this skill when designing interactive user flows, state representations, or pipeline wizards.

## Guidelines

### 1. Delivery Lifecycles & Pipeline States
- **Linear Progress Indicator**: Give users a clear map of their position in the pipeline (e.g. active animations for the current stage, green checks for completed ones).
- **Interactive Forms**: Highlight inputs on focus, use distinct descriptors, and validate fields instantly on submit.
- **Micro-interactions**: Incorporate transitions (`transition-all duration-300`) on all hoverable cards and buttons to make the UI feel alive.

### 2. State & Resolution Indicators
- **Descriptive Badges**: Use background-tinted, colored border badges for states (yellow for intake, cyan for PR review, emerald for shipped).
- **Empty States**: Render empty states with clean icon visuals and clear instructions on how to initiate progress.
